Coupon generator : Duplicating a single source coupon multiple times & generate multiple codes. Download all codes in CSV format too. None fancy/Get the job done type.

== Description ==

Simple but powerful coupon generator. Use with caution.
Creating multiple coupons with unique codes - from single source coupon.
Easy solution and better results, while paying nothing!
Was created because other similar alternatives - cost something.
Feel free to donate anything between nothing to something :)


== Installation ==

1. Upload and install plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. on your coupons list you will get new actions: 'Bulk Duplicate' | 'Bulk Delete'
4. use it wisely. 

== Frequently Asked Questions ==

= which types of coupons it support? =

any type. it will duplicate your selected coupon into multiple coupons with unique codes.

= how to delete all these generated codes? =

use "bulk delete" and just type the prefix used in all the generated coupons.
all coupons with the same prefix will be deleted.

= my client wants a list of all codes for his clients?.. =

After you 'bulk duplicate' your codes - don't forget to download the generated CSV file.
